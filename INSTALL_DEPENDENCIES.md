# Установка зависимостей

После клонирования проекта необходимо установить дополнительные зависимости для валидации:

```bash
cd server
npm install zod
```

Или если вы устанавливаете все зависимости заново:

```bash
cd server
npm install
```

**Важно:** Убедитесь, что `zod` установлен, так как он используется для валидации данных на сервере.
