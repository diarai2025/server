# API ключ добавлен

OpenAI API ключ был добавлен в файл `server/.env`.

## Что дальше?

1. **Перезапустите сервер** (если он запущен):
   - Остановите текущий процесс (Ctrl+C)
   - Запустите снова: `npm run dev`

2. **Проверьте статус**:
   ```bash
   curl http://localhost:3001/api/ai/status
   ```

   Должен вернуть:
   ```json
   {
     "available": true,
     "configured": true,
     "message": "OpenAI API настроен и готов к использованию"
   }
   ```

3. **Используйте AI функции**:
   - При создании кампании AI будет использовать GPT для генерации текстов
   - Рекомендации будут более умными и персонализированными
   - Тексты объявлений будут уникальными для каждой кампании

## Безопасность

⚠️ **Важно**: 
- Файл `.env` уже в `.gitignore` и не будет закоммичен
- НЕ делитесь API ключом публично
- Регулярно проверяйте использование ключа на platform.openai.com

## Если нужно изменить ключ

Просто отредактируйте строку `OPENAI_API_KEY` в файле `server/.env` и перезапустите сервер.
