# Быстрое исправление ошибки кошелька

## Проблема
Сервер возвращает ошибку 500 при попытке загрузить кошелек. Таблица `wallets` уже существует в базе данных, но Prisma Client не обновлен.

## Решение (2 команды)

### 1. Обновите Prisma Client:
```bash
cd server
npm run prisma:generate
```

### 2. Перезапустите сервер:
```bash
# Остановите текущий сервер (Ctrl+C), затем:
npm run dev
```

## Альтернатива (если npm не работает)

Если команда `npm run prisma:generate` не работает, попробуйте:

```bash
cd server
./node_modules/.bin/prisma generate
```

Или используйте скрипт:
```bash
cd server
./update-prisma.sh
```

## Проверка

После выполнения команд:
1. Откройте приложение в браузере
2. Кошелек должен загрузиться без ошибок
3. В консоли сервера должны появиться логи:
   - `[getWallet] Запрос на получение кошелька для пользователя: ...`
   - `[getWallet] Пользователь найден, ID: ...`
   - `[getWallet] Кошелек найден/создан`
